# Sistema de Gestão de Biblioteca

## Descrição do Projeto

Sistema simples de gestão de itens de biblioteca com cadastro, empréstimo e devolução. Desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

##  Objetivos de Aprendizagem

- ✅ Encapsulamento: Atributos privados (`__`) + getters/setters
- ✅ Herança: Superclasse `ItemBiblioteca` e subclasses `Livro` e `Revista`
- ✅ Polimorfismo: Métodos sobrescritos nas subclasses
- ✅ Organização: Código separado em arquivos com comentários
- ✅ Git/GitHub: Versionamento com commits claros e frequentes

##  Como Executar ✔

### Pré-requisitos
- Python 3.7 ou superior ✔
- Git (para versionamento) ✔

### Passos
1. Clone o repositório
2. Navegue até a pasta `biblioteca-poo`
3. Execute o comando:
```bash
python main.py
